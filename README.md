# Student Mental Health Support Platform

A responsive React-based mental health support platform for students.

## Features

- **Home Page**: Welcoming introduction with calming design
- **Resources**: Mental health articles, videos, and tips with search/filter
- **Therapy Sessions**: Form to schedule virtual therapy sessions with validation
- **Support Groups**: Community groups with simulated chat forum
- **Login**: Simple authentication interface
- **Admin Dashboard**: Manage therapy requests and add resources

## Design

- Calming blue/green color palette
- Fully responsive (mobile & desktop)
- Smooth animations and hover effects
- Clean, accessible interface

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Technologies Used

- React 18
- React Router DOM
- CSS3 (Flexbox/Grid)
- JavaScript ES6+
